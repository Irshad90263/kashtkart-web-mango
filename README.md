# 🪔 SKS Sandila Laddu – E-Commerce Website

A modern, responsive e-commerce website for traditional Indian laddus, built with **React**, **Tailwind CSS**, and **Context API**.  
Designed with a **Black & Gold luxury theme**, focused on simplicity, performance, and scalability.

---

## 🚀 Features

- 🛍️ Product listing (Laddus)
- 🛒 Add to Cart
- 💳 Checkout flow
- 🎨 Global Theme (Colors & Fonts via Context)
- 🌐 Google Fonts integration
- 📱 Fully responsive design
- ⚡ Fast build with Vite
- 🧩 Scalable folder structure (production-ready)

---

## 🧰 Tech Stack

- **Frontend:** React (JSX)
- **Styling:** Tailwind CSS
- **State Management:** Context API
- **Build Tool:** Vite
- **Fonts:** Google Fonts
- **Theme:** CSS Variables + Tailwind

---

## 🎨 Theme System

The project uses a **Theme Context** so that:
- Colors & fonts are defined once
- Changes apply across the entire project
- Easy future customization

### Theme Colors
| Name | Hex |
|-----|----|
| Primary (Black) | `#0B0B0B` |
| Secondary (Gold) | `#D4AF37` |
| Accent (Cream) | `#F5EEDC` |

### Fonts
- **Headings:** Playfair Display
- **Body:** Poppins
- **Accent / Buttons:** Montserrat
